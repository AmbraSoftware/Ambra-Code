# 🚀 INSTALAÇÃO RÁPIDA - Ambra Food Web

## 📋 PRÉ-REQUISITOS

- ✅ Node.js 18+ instalado
- ✅ Backend rodando em `http://localhost:3333` (ou configure `.env.local`)

---

## ⚡ INSTALAÇÃO (2 minutos)

### 1. Instalar Dependências

```powershell
cd C:\Users\Usuário\Documents\AmbraCode\apps\ambra-food-web
npm install
```

### 2. Configurar API URL (Opcional)

Se o backend está em IP diferente, edite `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://192.168.15.9:3333
```

### 3. Rodar Aplicação

```powershell
npm run dev
```

---

## ✅ PRONTO!

Acesse: `http://localhost:3002`

**No celular (mesma rede Wi-Fi):**

1. Descubra seu IP: `ipconfig`
2. Acesse no navegador do celular: `http://192.168.15.9:3002`
